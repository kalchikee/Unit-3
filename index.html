Choropleth Map & Coordinated Bar Chart

This interactive visualization demonstrates a choropleth map of US cities colored by population, with a coordinated bar chart showing the top 5 cities from each population classification. Both visualizations use the same quantile color scale for consistent data representation.

Key Features:

  • Choropleth Map: US cities displayed as colored circles based on population density
  • Coordinated Bar Chart: Top 5 cities from each classification shown as sorted bars
  • Quantile Classification: Data divided into 5 equal groups for meaningful comparison
  • Color Legend: Clear visual guide for interpreting population classifications
  • TopoJSON Integration: Efficient state boundaries using optimized geographic data
  • Responsive Design: Adapts seamlessly to different screen sizes
Data Sources & Technology:
US Cities Database (SimpleMaps) | TopoJSON Format | D3.js Library
